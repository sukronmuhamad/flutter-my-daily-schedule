# My Daily Schedule - Flutter App

Aplikasi pengingat jadwal harian sederhana yang dibangun dengan Flutter. Data tetap tersimpan meskipun aplikasi ditutup atau HP direstart.

## ✨ Fitur Utama
* **Local Persistence:** Menggunakan `shared_preferences` untuk menyimpan data secara permanen di memori HP.
* **Modern UI:** Tampilan bersih dengan integrasi Material Design.
* **Responsive:** Nyaman digunakan di berbagai ukuran layar Android.

## 🛠️ Teknologi yang Digunakan
* **Framework:** Flutter
* **Language:** Dart
* **Storage:** Shared Preferences
* **Tools:** VS Code, JDK 17

## 🚀 Cara Menjalankan
1. Clone repository ini.
2. Jalankan `flutter pub get`.
3. Jalankan `flutter run` pada emulator atau perangkat fisik.
